Possibility

Sample space ($\Omega$) is a set of all possible outcomes or results of a random experiment.

Points $\omega$ in $\Omega$ are called sample outcomes, realizations, or elements.

Subsets of $\Omega$ are called Events.

$e.g.$:

if $\Omega = {1,2,3,4,5,6}$, A= ‘Even result’=${2,4,6}$ is an event.

Let $S$ be a sample space. A probability on $S$ is a real-valued function $P$, $P:{\text{Events}}\to \mathbb{R}$.

satisfying:

  1. $P(A)≥0$ for any event $A$.

  2. $P(S)=1$.

  3. If $A1, A2, . . .$ are mutually exclusive events (m.e.e) then $P\left(\displaystyle \bigcup_{i=1}^{\infty} A_i\right) = \displaystyle \sum_{i=1}^{\infty} P(A_i)$

    whereby m.e.e. we mean $A_i∩A_j =\emptyset$ when $i\neq j$. ($\emptyset$ means empty set)

Basic properties of probability:

  1. $P(∅)=0$.

  2. $P(A’)=1−P(A)$.

  3. If $A\subset B$ then $P(A)≤P(B)$. In particular, if $B$ is $S$, we get $0 ≤ P(A) ≤ 1$ for any event $A$.

  4. Let $B_1,B_2,…$ be m.e.e. such that $S=\left(\displaystyle \bigcup_{i=1}^{\infty} B_i\right)$, that is, the $B_i$’s form a partition of $S$.

    Then for any event $A$, $P(A) = \displaystyle \sum_{i=1}^{\infty} P(A\cap B_i)$

  5. that for any event $B$, since $S = B \cup B’$ is a partition of $S$, then $P(A) = P(A \cap B) + P(A \cap B’)$.

    In particular, since $A \cap B’ = A \setminus B$ (They are two different ways to express {in A but not in B}), we have $P(A \setminus B) = P(A) − P(A \cap B)$.

  6. For any events $A$ and $B$, $P (A \cup B) = P (A) + P (B) − P (A \cap B)$.

  7. If $A,B$ are independent events of $\Omega \iff P(A \cap B) = P(A)\cdot P(B)$
