Apply Operations to an Array

You are given a 0-indexed array nums of size n consisting of non-negative integers.

You need to apply n - 1 operations to this array where, in the ith operation (0-indexed), you will apply the following on the ith element of nums:

  • If nums[i] == nums[i + 1], then multiply nums[i] by 2 and set nums[i + 1] to 0. Otherwise, you skip this operation.

After performing all the operations, shift all the 0's to the end of the array.

  • For example, the array [1,0,2,0,0,1] after shifting all its 0's to the end, is [1,2,1,0,0,0].

Return the resulting array.

Note that the operations are applied sequentially, not all at once.

Constraints:

  • 2 <= nums.length <= 2000
  • 0 <= nums[i] <= 1000

在python中, bool值False=0, True=1;

[],{},'',None,() ,0当这些数据结构为空时, bool(arg)==False, 在上面的解法就是运用了这一点, 判断列表中元素是否为0, 这样就把0全部移到最后面去啦; 如果不适用这个点,同样可以再次遍历列表, 进行判断, 再进行列表的增删操作

二叉树理解

树(Tree):

是存在n(n>=0)个节点的集合结构, 从视觉上看, 像是一棵圣诞树.

组成这棵圣诞树的每个元素称为节点(Node);

每一个树形结构中最顶端的节点称为根节点(Root), 根节点直接连接的节点为其的子节点;

从根节点到最尾端的子节点,每一层都是这颗树的层级(Level);

最大层级就是树的深度(Depth);

总体的树结构可以看成是很多子树(Subtree)的组合.

如图所示, 总体来看的话,这棵树深度为4, 根节点为A;而对于节点D,G,H,I所构成的子树而言,根节点为D,深度为2.

二叉树(Binary Tree):

树形结构的一种, 二叉树限定了每一个根节点下的子节点最多有2个,并且有严格的左右区分

二叉树用处

最初我是从排序算法中了解到这个结构, 检索或是插入数据时, 我们可以按一定的规则,比如根节点值<左节点值<右节点值构建一个二叉树结构, 对于指定的数据, 可以很快定位到他所处的位置; 机器学习中的决策树(decision tree)模型同样也是使用了二叉树结构

二叉树遍历

前序遍历:

根节点–>左节点–>右节点: 从视觉上来感知整个路径,就是从根节点出发,按逆时针方向遍历,每一个子树结构也同样遵循此规则

后序遍历:

左节点–>右节点–>根节点: 方向由树的底部通往顶部,优先左节点,再右节点,最后是根节点; 每一个子树结构按此规则遍历完成后, 再到同层级的子树结构

中序遍历:

左节点–>根节点–>右节点: 这个很好理解,想象将树的每一个节点都垂直投影到平面上,从左到右遍历

层次遍历:

与中序遍历类似, 按层级由左到右进行遍历
