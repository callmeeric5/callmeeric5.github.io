Abstract Data Types

Abstract Data Types (ADTs) An ADT is a tuple （T, U, O, P, A） where T is the name of the ADT we are defining. U is a set of （already defined） ADTs used in the definition of T. Notice that this implies a notion of a hierarchy of ADTs. O is a set of operation signatures that defines the syntax for the legal use of T objects.
