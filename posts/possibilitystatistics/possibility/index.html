Possibility#

Sample space ($\Omega$) is a set of all possible outcomes or results of a random experiment.

Points $\omega$ in $\Omega$ are called sample outcomes, realizations, or elements.

Subsets of $\Omega$ are called Events.

$e.g.$:

if $\Omega = {1,2,3,4,5,6}$, A= ‘Even result’=${2,4,6}$ is an event.

Let $S$ be a sample space. A probability on $S$ is a real-valued function $P$, $P:{\text{Events}}\to \mathbb{R}$.

satisfying:

  1. $P(A)вүҘ0$ for any event $A$.

  2. $P(S)=1$.

  3. If $A1, A2, . . .$ are mutually exclusive events (m.e.e) then $P\left(\displaystyle \bigcup_{i=1}^{\infty} A_i\right) = \displaystyle \sum_{i=1}^{\infty} P(A_i)$

    whereby m.e.e. we mean $A_iвҲ©A_j =\emptyset$ when $i\neq j$. ($\emptyset$ means empty set)

Basic properties of probability:#

  1. $P(вҲ…)=0$.

  2. $P(A’)=1вҲ’P(A)$.

  3. If $A\subset B$ then $P(A)вүӨP(B)$. In particular, if $B$ is $S$, we get $0 вүӨ P(A) вүӨ 1$ for any event $A$.

  4. Let $B_1,B_2,…$ be m.e.e. such that $S=\left(\displaystyle \bigcup_{i=1}^{\infty} B_i\right)$, that is, the $B_i$вҖҷs form a partition of $S$.

    Then for any event $A$, $P(A) = \displaystyle \sum_{i=1}^{\infty} P(A\cap B_i)$

  5. that for any event $B$, since $S = B \cup B’$ is a partition of $S$, then $P(A) = P(A \cap B) + P(A \cap B’)$.

    In particular, since $A \cap B’ = A \setminus B$ (They are two different ways to express {in A but not in B}), we have $P(A \setminus B) = P(A) вҲ’ P(A \cap B)$.

  6. For any events $A$ and $B$, $P (A \cup B) = P (A) + P (B) вҲ’ P (A \cap B)$.

  7. If $A,B$ are independent events of $\Omega \iff P(A \cap B) = P(A)\cdot P(B)$

Marginalisation#

$P(x=x_i) =\displaystyle \sum_{j=1}^n P(x=x_i,y=y_j)$

Mean, variance, standard deviation#

$\mu = E[x] =\displaystyle \sum_{i=1}^nx_iP_i$

$Var(x) = E[(x-\mu)^2]=E[(x-E[x])^2] = E[x^2-2xE[x]+(E[x])^2]=E[x^2]-2E[x]E[x]+(E[x])^2=E[x^2]-(E[x])^2=E[x^2]-\mu^2$

Bernoulli Distribution#

$Bern(x|\mu)=\mu^x(1-\mu)^{1-x}$

$Var(x)=\mu(1-\mu)$

Covariance#

$Cov(x,y)=E[(x-\mu_x)(y-\mu_y)]$

N.B.:

  1. If $Cov(x,y)=0$, just means there is no linear relationship between $x$ and $y$! It doesn’t mean $x$ and $y$ are independent. eg:$x^2+y^2 = 1$
  2. If $Cov(x,y)>0$, when $x$ increase, y increase
  3. if $Cov(x,y)<0$, when x decrease, y decrease

Conditional possibility#

If $A,B$ are independent events, $P(A \cap B)= P(A)\cdot P(B)$

If $A,B$ are not necessarily independent events, $P(A \cap B)=P(B|A)\cdot P(A)$ also can be written as $P(A|B) = \frac {P(A \cap B)} {P(B)}$

law of total possibility#

$P(A)=P(A|B_1)P(B_1)+P(A|B_2)P(B_2)+…+P(A|B_n)P(B_n)$

If $A,B$ are independent events, $P(A | B)= \displaystyle \frac {P(A)\cdot P(B)}{P(B)}= P(A)$

Dividing both sides by $P(B)$ gives $рқ‘ғ(рқҗҙ|рқҗө) =\frac {рқ‘ғ(рқҗө|рқҗҙ)рқ‘ғ(рқҗҙ)} {рқ‘ғ(рқҗө)} $
