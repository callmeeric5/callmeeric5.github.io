Diagonalisation#

An n × n matrix A is diagonalisable if it is similar to a diagonal matrix: that is if there exists an invertible n × n matrix C and a diagonal matrix D such that $A= CDC^{−1}$.

Compute powers of a diagonalisable matrix. If $A= CDC^{−1}$, where $D$ is a diagonal matrix, then $A^n = CD^nC^{−1}$:

$$ A = C \begin{bmatrix} x &0 &0\\\\ 0 &y &0\\\\ 0 &0 &z \end{bmatrix} C^{-1} \iff A^n = C \begin{bmatrix} x^n &0 &0\\\\ 0 &y^n &0\\\\ 0 &0 &z^n \end{bmatrix} C^{-1} $$
