Optimization#

we are looking for the min or max of a function on a specific domain. Generally, optimization problems > convex problems > linear programming problems

Convex problems#

An optimization problem is convex to find $min(f(x)), x\in D$, $D$ is a convex set and $f$ is a convex function. In brief, finding a local min or global min is enough

Convex sets#

Is it always possible to draw a line between 2 points of my set, all the lines are also included in the set

Definition: $S$ is a shape in $R^2$, S is convex if $\forall x,y\in S,\forall t \in (0,1), tx+(1-t)y \in S$

Convex functions#

A function $f:D \in R^n \to R$ is convex if

  • $D$ is a convex set

  • $f(tx+(1-t)y) \le tf(x)+(1-t)f(y),\forall t \in(0,1),\forall x,y\in D$

means that the curve representing $f$ is always below the line joining 2 points of the curve

**N.B.: **

  • in a convex function, a local min is the global min
  • In a convex function, the gradient is an incremental ascent

Linear programming problem#

$-f(x) = f(-x)$

$f(\lambda x)= \lambda f(x)$

$$ \begin{cases} ax+by =c_1\\\\ mx+by = c_2 \end{cases} $$

$a,b,c_1,c_2$ are constraints of this linear equation.

The area contributed by the feasible solutions(where all constraints are satisfied) is called the feasible region.

Simplex method#

If we deal with an LPP which is a maximization problem with a non-negativity constraint, we can use the simplex method.

The idea of the simplex method is finding all solutions, the vertex of the graph

To solve a linear programming problem in standard form, use the following steps.

  1. Convert each inequality in the set of constraints to an equation by adding slack variables.
  2. Create the initial simplex tableau.
  3. Locate the most negative entry in the bottom row. The column for this entry is called the entering column. (If ties occur, any of the tied entries can be used to determine the entering column.)
  4. From the ratios of the entries in the “b-column” with their corresponding positive entries in the entering column. The departing row corresponds to the smallest nonnegative ratio (If all entries in the entering column are 0 or negative, then there is no maximum solution. For ties, choose either entry.) The entry in the departing row and the entering column is called the pivot.
  5. Use elementary row operations so that the pivot is 1, and all other entries in the entering column are 0. This process is called pivoting.
  6. If all entries in the bottom row are zero or positive, this is the final tableau. If not, go back to Step 3.
  7. If you obtain a final tableau, then the linear programming problem has a maximum solution, which is given by the entry in the lower-right corner of the tableau.

Degeneracy#

We have a case of degeneracy if one basic variable is equal to zero.

Degeneracy can lead to cycling (=repeating the same steps in the implementation of the simplex) and an infinite loop (not all the time)

From the geometric explanation, that means more than two constraint lines are intersecting at the same point. So there must exist one redundant constraint. The objective function, is parallel to one constraint (non-redundant one)
