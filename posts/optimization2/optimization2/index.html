Critical Value#

$x = c$ is a critical value if $f’(c) = 0$ or $f’(c)$ doesn’t exist

N.B.: If $f’(c) = 0$ doesn’t imply $x = c$ is a critical value

Second derivative test#

If $x=c$ is a critical value of $f(x)$, such that $f’(c)=0$ then $x=c$

$f’’(c)<0$ the graph of the function is concave, A-shape

$f’’(c)>0$ the graph of the function is convex, U-shape

$f(c)$ is a local max (the maximum around one peak) if $f’’(c)<0$

$f(c)$ is a local min (the minimum around one valley) if $f’’(c)>0$

$f(c)$ may be a local max or min or neither if $f’’(c)=0$

Newton’s method#

$x_{n+1} = x_n - \dfrac {f(x_n)} {f’(x_n)}$

Extreme value theorem:#

if $f$ is a continuous function defined on the closed interval $D = [a,b]$, it has an absolute max and absolute min on $[a,b]$

Chain rule of partial derivatives(Total derivative)#

$\dfrac {dz} {dt} = \dfrac {\partial z}{\partial x}\dfrac {dx} {dt}+\dfrac {\partial z}{\partial y}\dfrac {dy} {dt}$

also, denoted as $df = \dfrac {\partial f(x,y)} {\part x}dx + \dfrac {\partial f(x,y)} {\part y}dy$

Tylor series for 2 variable#

If $f$ is a function where partial derivatives exist near the point $(a,b)$, for $(x,y)$ close to $(a,b)$ we can write:

$f(x,y) \approx f(a,b) + f_x(a,b)(x-a) + f_y(a,b)(y-b)$

$f(x,y) \approx f(a,b) + f_x(a,b)(x-a) + f_y(a,b)(y-b)+\frac 1 2 f_{xx}(a,b)(x-a)^2+f_{xy}(x-a)(x-b)+\frac 1 2 f_{yy}(a,b)(x-b)^2$
